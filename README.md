# Ejemplo de POO: Dispositivo Inteligentes

Este proyecto es un ejemplo de **POO en java** usando **clases abstractas** e **interfaces**.  
Es muy basico y solo simula un dispositivo inteligente (un foco xd) que se puede encender, apagar, conectarse a Wi-Fi y hacer acciones como ajustar el brillo.
